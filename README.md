# opbio - Work in Progress

opbio is a feature-rich social platform built with **Next.js**, **React**, **TypeScript**, **JavaScript**, **Prisma**, and **MySQL**. This project is designed to allow users to create personal profiles, similar to Discord, and engage with the community through forums and profile comments. The authentication system, including OTP verification and email updates, has been built entirely from scratch.

## üöÄ Features

- **Custom User Profiles** ‚Äì Every user gets a personalized profile, similar to Discord.
- **Forum System** ‚Äì Users can create and participate in discussions.
- **Profile Comments** ‚Äì Engage with others by commenting on their profiles.
- **Secure Authentication** ‚Äì Custom-built authentication system with OTP verification.
- **Email Change with OTP** ‚Äì Users can update their email securely.
- **Modern Tech Stack** ‚Äì Built with Next.js, React, TypeScript, JavaScript, Prisma, and MySQL.

## üîß Installation & Setup

1. Clone the repository:

   ```sh
   git clone https://github.com/Teeermi/opbio.git
   cd opbio
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

3. Set up environment variables:

   - Create a `.env` file in the root directory
   - Add the following environment variables:
     ```env
     DATABASE_URL=your_mysql_database_url
     SESSION_SECRET=your_session_secret
     RESEND_API_KEY=your_resend_api_key
     ```

4. Run the development server:

   ```sh
   npm run dev
   ```

5. Build and start the production server:
   ```sh
   npm run build
   npm run start
   ```

## üíª Commands

- **Start dev server**: `npm run dev`
- **Build production**: `npm run build`
